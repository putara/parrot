# Introduction
A CPU monitor app inspired by [PPCpuMon](https://github.com/tomozh/PPCpuMon).

## How to use
1. Download some [GIFs](https://cultofthepartyparrot.com/) in the "gifs" folder
1. Start parrot.exe
